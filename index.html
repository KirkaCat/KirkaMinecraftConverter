Kirka x Minecraft Map Converter

Convert a Kirka map to a Minecraft schematic file or a schematic file to a Kirka map code.
Made for Minecraft 1.20

Note:
Everything was made and tested using 1.20.6. Use other 1.20 versions at your own risk, though it should be fine :)


Convert:

  • Kirka ➜ Minecraft
  • Minecraft ➜ Kirka

Minecraft Mods:
  • Fabric Loader Installer
  • Fabric API (direct)
  • Sodium (direct) - A mod that optimizes Minecraft, drastically increasing performance.
  • WorldEdit (direct)

Resources:
  • Kirka x Minecraft Resource pack
  • Kirka Block Palette Schematic - Load and paste this into your Minecraft world using WorldEdit. Don't forget to use -a in your command.
  • Default Kirka Textures
    • 0.42+
    • 0.41+
  • Kirka block pallete map code
  • Kirka Map Codes 02 June 2025 from the official Kirka Discord

  • Marker Utils Preview Schematic - A command block. Load and paste this into your Minecraft world using WorldEdit.
